# Tools Manifest

> This file lists all available tools in the `tools/` directory.

## Memory Tools

| Tool | Description |
|------|-------------|
| `memory/memory_db.py` | Manages SQLite database connections and operations for memory and activity tracking. |
| `memory/memory_read.py` | Reads from the memory database and logs. |
| `memory/memory_write.py` | Writes to the memory database, logs, and MEMORY.md. |
| `memory/embed_memory.py` | Generates embeddings for memory entries (if applicable). |
| `memory/hybrid_search.py` | Performs hybrid search on memory entries. |
| `memory/semantic_search.py` | Performs semantic search on memory entries. |
